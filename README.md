# 🌐 Projeto de Rede - Packet Tracer

Simulação de uma rede cliente-servidor desenvolvida no Cisco Packet Tracer.

## 📌 Estrutura da rede
- 1 Servidor
- 1 Roteador
- 1 Switch
- 3 Computadores

## ⚙️ Funcionalidades
- Distribuição automática de IP (DHCP)
- Resolução de nomes (DNS)
- Servidor web ativo
